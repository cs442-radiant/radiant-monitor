# Radiant Monitor

## How to run
1. Install [meteor](https://www.meteor.com/)
2. Install [npm](https://www.npmjs.com/) and [node](https://nodejs.org/)
3. $ meteor npm install
4. $ meteor run --settings settings.json
5. Check http://localhost:3000/
